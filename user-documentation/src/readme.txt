Files with the extension mbl are to be converted to marbles images.

See https://bitbucket.org/achary/rx-marbles/src/master/ and https://bitbucket.org/achary/rx-marbles/src/762c9519e58f4cb914c9187c0b3019d5c665b933/docs/syntax.md

Install with 'pip3 install rxmarbles'
Simply run the script generate-marbles.sh to generate all of them.
